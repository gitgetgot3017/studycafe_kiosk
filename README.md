# STUDYCAFE KIOSK
#### 스터디카페에서 입퇴실을 하거나 이용권을 구매할 때 특정 앱을 사용하고 있는데, 사용하면서 개선되거나 추가되었으면 좋겠다고 생각한 기능들이 있었고, 이를 온전히 반영한 프로젝트를 만들고 싶었습니다.
<br>

### ✔ 기술 스택
- Java 17
- SpringBoot 3.3.4
- JPA
- MySQL
- react
<br>

### ⚙️ 모듈
- 회원(Member)
- 상품(Item)
- 주문(Order)
- 좌석(Seat)
- 쿠폰(Coupon)
- 이용권(Subscription)
- 입퇴실(UsageStatus)
- 투표(Vote)
- 게시판(Post)
<br>
